# Faaa

Faaa is an abbreviation of Function-as-a-agent.

## Overview

The target of faaa is to provide a fresh new interface for implementing multi-agent LLM applications. The initial word "function" in faaa indicates that our package can transport any function to an LLM function call with only a few lines of code.

## Features

- **Function Transport**: Easily convert any function to an LLM function call.
- **FastAPI Interface**: faaa also provides a FastAPI interface to help users build their API applications.
